# Kishlay Choudhary Portfolio

High-impact personal portfolio built with Next.js 14, Tailwind CSS, Framer Motion, and OpenRouter.

## Setup
1. Clone repo
2. npm install
3. Copy .env.local.example to .env.local
4. Add your OpenRouter API key (free at openrouter.ai)
5. npm run dev

## Tech
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- OpenRouter API (chatbot)
